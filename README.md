# Telegram Bot с веб-панелью администратора

## Функционал
- Пользователь пишет в бота.
- Сообщения сохраняются в базу SQLite.
- Веб-панель (`http://localhost:8000`) показывает всех пользователей и их сообщения.
- Админ может удалить отдельное сообщение или всю переписку.
- При удалении бот удаляет сообщение у пользователя через Telegram API.

## Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте `.env` файл:
   ```
   BOT_TOKEN=<токен вашего бота>
   ADMIN_PASSWORD=admin
   ```
3. Запустите:
   ```bash
   python app/main.py
   ```
